# Kindle highlights to Obsidian Markdown notes, written in Go.

## How to launch

```
go run main.go --input <path to My Clippings.txt> --output <path to output folder>
```

## Tested device

- Amazon Kindle Paperwhite 5th Generation (EY21)
- Fedora Linux
